Questão técnica

O Tenant não tem relação direta com Lease no schema atual. Para mostrar dados da locação ativa, precisaremos buscar os leases e cruzar com os tenants no frontend (via responsible_tenant_id ou via a lista tenants[] do lease). Os dados contract_signed, interfone_configured, contract_generated, pdf_path já existem no Lease.
